# Python Intro

## Setup

TERMINAL 

Create an environment
```
$ mkdir myproject
$ cd myproject
$ python3 -m venv venv
```

Activate your environment 
```
$ . venv/bin/activate
```

Use the green 'Play' button in the top right to run code and see the prints in the console.
